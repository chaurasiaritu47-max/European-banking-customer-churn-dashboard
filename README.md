# European-banking-customer-churn-dashboard
This project analyzes customer churn patterns in a European bank using Python, Streamlit, and Plotly. Interactive filters, KPIs, and visualizations help identify high-risk customer segments based on geography, balance, age, and engagement to support better customer retention strategies.
